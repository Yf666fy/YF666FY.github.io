004pycharm快捷键

发表于2024-11-21|更新于2024-11-21
|总字数:320|阅读时长:1分钟|浏览量:

基本操作

File: Alt + F1 打开文件
New File: Alt + Insert 创建新文件
Close: Ctrl + F4 关闭当前文件
Save: Ctrl + S 保存当前文件
Save All: Ctrl + Alt + S 保存所有更改的文件

编辑操作

Undo: Ctrl + Z 撤销
Redo: Ctrl + Shift + Z 重做
Cut: Ctrl + X 剪切
Copy: Ctrl + C 复制
Paste: Ctrl + V 粘贴
Delete: Delete 删除
Select All: Ctrl + A 全选
Find: Ctrl + F 查找
Find Next: F3 查找下一个
Find Previous: Shift + F3 查找上一个
Replace: Ctrl + R 替换
Find Usages: Alt + F7 查找使用
Replace in Files: Ctrl + R 在文件中替换

代码导航

Go to Class: Ctrl + N 跳转到类
Go to File: Ctrl + Shift + N 跳转到文件
Go to Symbol: Alt + F12 跳转到符号
Go to Line: Ctrl + G 跳转到行
Back: Alt + Left 后退
Forward: Alt + Right 前进

代码操作

Comment/Uncomment: Ctrl + / 切换注释
Auto-Indent Lines: Tab 自动缩进
Unindent Lines: Shift + Tab 取消缩进
Reformat Code: Ctrl + Alt + L 格式化代码
Optimize Imports: Ctrl + Alt + O 优化导入
Surround With: Ctrl + Alt + T 包围代码（如 if 语句）

调试和运行

Run: Shift + F10 运行程序
Debug: Shift + F9 开始调试
Stop: Ctrl + F2 停止程序
Step Over: F8 单步跳过
Step Into: F7 单步进入
Step Out: Shift + F8 单步跳出

版本控制

Commit: Ctrl + K 提交
Push: Ctrl + Shift + K 推送
Pull: Ctrl + T 拉取
